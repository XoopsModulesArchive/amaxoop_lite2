Amaxoopシリーズについて

当モジュールはフリーです。当モジュールは原則としてGPLに基づいて配布されています。
ただし、下記記載内容に関しては了解＆合意の上、御使用下さい。

使用方法については、何らかの形で提供しますが、基本的には「習うより慣れろ」と思ってください。
ひとえに作者が不精であるためですが、御質問は当方サイトでお受けします。

【商品情報】
当モジュールにおける商品情報はAmazonECSへのアクセスにより、データを取得します。
商品が入手できない場合や画像が無い場合には表示を行わない場合があります。

【Amazonへのリンク】
当モジュールにより生成されるAmazonへのリンクは下記となります。
  • ユーザが登録の際にアソシエイトIDを入力した場合、リンクの半分がユーザ、半分がサイト管理者のIDになります。
  • 各リンクのうち、5%が原作者のアソシエイトIDになります。御容赦下さい。
    実際の売上(報酬)に結び付くかどうかは神のみぞ知る。あくまでもリンク(トラフィック)の5%です。
  • 作成にあたり、相当な労力を費やしているため、これをねぎらって頂く意味です。
    モジュールを編集した場合やその再配布の際には、上記の5%ルール部分は変更しないで下さい。
  • Amazonへのリンクは、クリック回数をカウントするため、当モジュール内の別ページを経由する場合があります。
    リダイレクトとなりますが、Amazonその他に不要な情報を送ることはありません。

【注意事項】
  • マルチバイト関数を使っています。
    AmazonへのアクセスがUTF-8であり、XOOPSではEUC-JPを用いているため、この変換を行っています。
  • デバッグモードでは、「Undefined index」等のNOTICEが出ることがあります。
    申し訳ありませんが、これらには目をつぶってください。

【使用上の免責事項及び条件】
当モジュールを使用するにあたり、万が一不都合や不利益が発生しても、原作者はその責任を負いません。
  • バグや問題を発生させた原因の特定と解決に関して、原作者は自らが合理的と判断する範囲でこれらへの対応を行います。
    ただし、その対応の中には、金品等によるものは含みません。
  • 無責任に見えますが、お約束です。これらの不都合は、原作者としては発生しないものと考えております。

【あとがき】
アソシエイトサポートモジュールを配布しているだけあって、アソシエイト参加者しか作者のサイトを訪問してきません。
そのせいもあってか、作者のサイトからAmazonを訪問し商品を購入する方はほとんどさっぱりいらっしゃいません。
てなことでもあるし、モジュール作るのもこれでなかなか大変なので、トラフィックの5%だけは、どうぞ御容赦下さい。
